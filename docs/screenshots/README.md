# Screenshots do CFS Spool

Adicione as imagens abaixo nesta pasta para que aparecam automaticamente nos READMEs.

## Screenshots do Aplicativo

| Arquivo | O que capturar |
|---------|----------------|
| `app-main-screen.png` | Abra o CFS Spool e preencha o formulario com dados de exemplo (material, cor, etc). Tire um print da janela inteira mostrando todos os campos preenchidos. |
| `app-color-picker.png` | Clique no seletor de cores para abrir o popover com a paleta de 35 cores e o espectro. Tire um print mostrando o popover aberto. |

## Screenshots de Instalacao no macOS

| Arquivo | O que capturar |
|---------|----------------|
| `macos-gatekeeper-blocked.png` | Tente abrir o CFS Spool pela primeira vez (duplo clique). O macOS mostra um dialogo dizendo que o app "nao pode ser aberto porque nao foi possivel verificar o desenvolvedor", com os botoes "Cancelar" e "Mover para o Lixo". Tire um print desse dialogo. **Importante**: clique em "Cancelar", nao em "Mover para o Lixo". |
| `macos-privacy-security.png` | Apos o bloqueio, abra **Ajustes do Sistema → Privacidade e Seguranca**. Role ate a secao "Seguranca" onde aparece a mensagem sobre o CFS Spool com o botao **"Abrir Mesmo Assim"**. Tire um print dessa tela. |

## Formato Recomendado

- **Formato**: PNG
- **Largura**: 600-800px (para boa visualizacao no GitHub)
- **Retina**: Se possivel, capture em resolucao 2x e redimensione para melhor nitidez
